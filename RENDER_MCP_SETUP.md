# Render MCP setup (Cursor Cloud Agents)

Use this guide to connect Render MCP and set your workspace so the agent can inspect deploy logs, services, and environment variables.

---

## Why workspace selection matters

Render MCP tools (`list_services`, `list_logs`, etc.) only work **after** you select which Render workspace (team/personal account) to use.

---

## Step 1 — Create a Render API key

1. Open [Render Dashboard → Account Settings → API Keys](https://dashboard.render.com/u/settings#api-keys)
2. Click **Create API Key**
3. Copy the key (shown once)

---

## Step 2 — Connect Render MCP in Cursor

### Option A — Cloud Agents (recommended for this repo)

1. Open [cursor.com/agents](https://cursor.com/agents)
2. Click the **MCP** dropdown (top of agents page)
3. Add **Render** MCP server:
   - URL: `https://mcp.render.com/mcp`
   - Header: `Authorization: Bearer YOUR_RENDER_API_KEY`
4. Save and ensure Render shows as **enabled**

### Option B — Cursor Desktop (local)

Add to `~/.cursor/mcp.json`:

```json
{
  "mcpServers": {
    "render": {
      "url": "https://mcp.render.com/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_RENDER_API_KEY"
      }
    }
  }
}
```

Restart Cursor after saving.

See `.cursor/mcp.json.example` in this repo for a template.

---

## Step 3 — Set your Render workspace

After MCP is connected, start a **new** agent chat and send:

```
Set my Render workspace to YOUR_WORKSPACE_NAME
```

Replace `YOUR_WORKSPACE_NAME` with your Render team or personal workspace name (as shown in the Render dashboard top-left).

### Don't know your workspace name?

Ask the agent:

```
List my Render workspaces
```

Then:

```
Set my Render workspace to [name from the list]
```

---

## Step 4 — Verify it works

Ask:

```
List my Render services
```

or

```
Get the latest deploy logs for school-portal
```

If MCP is connected and workspace is set, the agent can read your Render deployment status.

---

## Troubleshooting

| Problem | Fix |
|---------|-----|
| "MCP server does not exist: render" | Render MCP not enabled for this agent run — configure in cursor.com/agents MCP dropdown and start a **new** chat |
| Agent asks to select workspace | Run `List my Render workspaces` then `Set my Render workspace to ...` |
| Invalid API key | Create a new key in Render Account Settings |
| Wrong workspace | List workspaces again and switch |

| Variable | Value |
|----------|-------|

| `SESSION_SECRET` | Auto-generated by Render |
| `ADMIN_PASSWORD` | Check Render Environment tab after deploy |
